# 🌾 Farm Loan Risk Predictor

## 📌 Overview
The **Farm Loan Risk Predictor** is a machine learning-based project designed to evaluate the risk associated with granting loans to farmers. It analyzes multiple agricultural and financial factors to predict whether a farmer is likely to repay a loan or default.

This system helps financial institutions, banks, and microfinance organizations make **data-driven lending decisions**, reducing financial risk and improving loan approval efficiency.

## 🚀 Features
- 📊 Predicts loan repayment risk (Low / Medium / High)
- 🌦️ Considers agricultural factors like weather and crop yield
- 💰 Includes financial data such as income and past loan history
- 🤖 Machine learning-based prediction model
- 📈 Improves decision-making for lenders
- 🧑‍🌾 Supports farmer-focused financial inclusion

## 📊 Dataset Description
The dataset includes the following features:

- Farmer’s Income  
- Land Size  
- Crop Type  
- Crop Yield  
- Loan Amount  
- Previous Loan History  
- Weather Conditions  
- Soil Quality
